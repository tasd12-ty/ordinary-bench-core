# 信念重建判定标准

## 概述

信念重建（Belief Reconstruction）从 VLM 的回答中提取空间约束，用梯度优化求解 2D 位置，再评估这些位置是否自洽。整个流程分为三级门槛：**有解** → **可渲染 SVG** → **重建成功**。

## 第一级：是否有解（solver 能否运行）

VLM 的 QRR 回答（距离比较）会被构建为偏序图（distance poset）。如果偏序图中存在**传递性矛盾环**（例如 A>B, B>C, C>A），则直接判定 `infeasible`，solver 不运行，`positions` 为空。

**通过条件：QRR 回答无传递性矛盾环**

统计：约 82% 的场景因存在矛盾环而在此步被淘汰。

## 第二级：是否可渲染 SVG

只要 solver 运行并输出了非空的 `positions`（即通过了第一级），且存在 GT 场景数据用于对比，就会生成 SVG。

**SVG 包含三个面板：**
- **Ground Truth**：真实物体位置
- **Reconstructed**：solver 求解的位置（模型的"空间信念"）
- **Overlay**：经 Procrustes 对齐（含镜像反射）后的叠加对比，箭头表示偏差

**注意：有 SVG 不等于重建成功。** 很多场景虽然没有传递性矛盾环（通过了第一级），但模型的回答之间仍存在大量不一致——不存在任何一组 2D 位置能同时满足所有回答。这些场景的 CSR 很低，SVG 只是 solver 在互相矛盾的约束下做出的折中可视化。

**示例：有 SVG 但重建失败（CSR 低）**

假设场景有 A、B、C、D 四个物体，模型回答了以下 5 条约束：

| # | 约束类型 | 模型回答 | 含义 |
|---|---------|---------|------|
| 1 | QRR | dist(A,B) < dist(C,D) | A 和 B 比 C 和 D 更近 |
| 2 | QRR | dist(C,D) < dist(A,C) | C 和 D 比 A 和 C 更近 |
| 3 | QRR | dist(A,C) < dist(A,B) | A 和 C 比 A 和 B 更近 |
| 4 | TRR | B 在 A→C 的 3 点方向 | B 在 A 看向 C 的右侧 |
| 5 | TRR | B 在 A→C 的 9 点方向 | B 在 A 看向 C 的左侧 |

**分析：**
- 约束 1-2-3 构成距离矛盾：`dist(A,B) < dist(C,D) < dist(A,C) < dist(A,B)`，形成环，但如果差距较小，solver 仍会尝试折中求解
- 约束 4 和 5 直接矛盾：B 不可能同时在 A→C 的左右两侧

Solver 会找到一组位置使总 loss 最小（比如满足约束 1、2、4），但反向验证时：
- 5 条中只有 3 条被满足 → **CSR = 0.60**，低于 0.95 → 判定为 `infeasible`
- 但仍然有非空 positions → **会生成 SVG**

这说明模型没有形成一致的空间信念，只是对不同问题给出了互相矛盾的答案。

## 第三级：是否重建成功（feasible）

重建成功需要同时满足两个 CSR（约束满足率）阈值：

```
feasible = (CSR_QRR >= 0.95) AND (CSR_TRR >= 0.95)
```

### CSR_QRR（距离约束满足率）

用重建位置反算每对物体的距离，检查是否与模型回答的比较符（`<`、`>`、`~=`）一致。

- 用 `tau=0.10` 的比率容差判定 `~=`
- 在镜像反射下不变（距离不受反射影响）

### CSR_TRR（方向约束满足率）

用重建位置反算钟面方向角，检查是否与模型回答的小时数一致。

- 评估时同时尝试两种手性（原始 + y 轴镜像反射），取较优值
- **solver 求解时使用象限级容差（±45°）**：只要求方向大致正确（同一象限），不苛求精确到小时
- **CSR 评估时根据约束来源使用不同容差**：
  - `level="hour"`：±15°（精确小时匹配）
  - `level="quadrant"`：±45°（象限匹配）

### 成功后的细分状态

| 状态 | 条件 | 含义 |
|------|------|------|
| `single_mode` | CSR ≥ 0.95, K_geom=1, spread ≤ 0.10 | 唯一解，最强信念 |
| `multimodal` | CSR ≥ 0.95, K_geom > 1 | 多个等价布局，约束不足以唯一确定 |
| `underconstrained` | CSR ≥ 0.95, spread > 0.10 | 解不唯一，约束较松 |
| `infeasible` | CSR < 0.95 或有矛盾环 | 回答互相矛盾，无法自洽 |

## SVG 可视化说明

以下以一张实际 SVG（场景 n04_000080，模型 gemini-3_1-pro-preview_multi_view）为例：

```
标题栏：Scene: n04_000080 | gemini-3_1-pro-preview_multi_view
指标栏：CSR_QRR=1.000 | CSR_TRR=0.958 | NRMS=0.1213 | τ=0.600 | K=1
```

### 三个面板

| 面板 | 内容 | 说明 |
|------|------|------|
| **Ground Truth**（左） | 真实 3D 场景的俯视 2D 投影 | 物体的真实空间位置，作为参照基准 |
| **Reconstructed**（中） | solver 从模型回答求解的 2D 位置 | 这就是模型的"空间信念"——它认为物体应该在哪里 |
| **Overlay**（右） | GT 与重建的叠加对比 | 实心 = GT 位置，虚线半透明 = 重建位置（经 Procrustes 对齐后），彩色箭头 = 偏差方向和大小 |

### 标题栏指标含义

**`CSR_QRR=1.000`** — 距离约束满足率（Constraint Satisfaction Rate for QRR）

模型对所有距离比较问题（"A 和 B 谁离 C 更近？"）的回答中，100% 能被重建位置同时满足。说明模型的距离判断完全自洽。

- 计算方式：用重建位置算出每对物体的实际距离，检查比较符（`<`/`>`/`~=`）是否与模型回答一致
- 范围：0~1，≥ 0.95 为自洽
- 0.0 表示 solver 未运行（有矛盾环）

**`CSR_TRR=0.958`** — 方向约束满足率（Constraint Satisfaction Rate for TRR）

模型对所有钟面方向问题（"C 在 A→B 的几点方向？"）的回答中，95.8% 能被重建位置满足。说明方向判断基本自洽，少量误差。

- 计算方式：用重建位置算出实际钟面角度，检查与模型回答的角度差是否在容差内
- 评估时同时尝试原始和 y 轴镜像两种手性，取较优值
- 范围：0~1，≥ 0.95 为自洽

**`NRMS=0.1213`** — 归一化均方根误差（Normalized Root Mean Square Error）

重建位置经最优相似变换（旋转+缩放+平移+镜像反射）对齐到 GT 后，平均位置误差占场景空间范围的 12.1%。

- 计算方式：Procrustes 对齐（allow_reflection=True）后的 RMS 除以 GT 的空间跨度
- 范围：0+ ，越低越好
- 0.0 = 与 GT 完全重合；0.05 = 偏差约为场景大小的 5%；> 0.3 = 偏差较大
- Overlay 面板标题中的 `NRMS=0.121` 即为此值

**`τ=0.600`** — Kendall tau 秩相关系数

重建位置的成对距离排序与 GT 成对距离排序的秩相关。

- 计算方式：取所有物体对的距离，分别在重建和 GT 中排序，计算 Kendall tau
- 范围：-1~1
- 1.0 = 距离排序完全一致；0.0 = 无相关；-1.0 = 完全反转
- 天然不受相似变换影响（包括镜像反射）

**`K=1`** — 几何模态数（K_geom）

10 次多重启优化收敛到几个本质不同的解。

- K=1：所有重启收敛到同一布局，约束唯一确定了空间结构（最强信念）
- K>1：存在多个满足约束的不同布局（约束不足以唯一确定）

### Overlay 面板图例

| 元素 | 含义 |
|------|------|
| 实心图形 + 标签 | GT（真实）位置 |
| 虚线半透明图形 | 重建位置（经 Procrustes 对齐后） |
| 彩色箭头 | 从 GT 到重建的偏差（Distortion），箭头越短越准确 |

### 指标之间的关系

```
CSR（自洽性指标）          NRMS / τ（准确性指标）
─────────────────          ────────────────────
衡量模型回答的内部一致性      衡量重建位置与真实位置的偏差
不需要 GT                   需要 GT 作为参照
CSR 高 → 模型有一致的        NRMS 低 → 模型的信念
         空间信念                     接近真实场景
CSR 高但 NRMS 高 →          CSR 低但 NRMS 低 →
模型自洽但与现实不符          模型部分正确但互相矛盾
```

## 相似变换不变性

重建结果仅在**相似变换**（平移 + 旋转 + 缩放 + 镜像反射）意义下确定。这意味着重建出的布局可能相对于 GT 有旋转、缩放或镜像翻转，但空间结构（谁离谁近、谁在谁的哪个方向）是一致的。

各指标的不变性：

| 指标 | 平移 | 旋转 | 缩放 | 镜像反射 | 处理方式 |
|------|:----:|:----:|:----:|:--------:|---------|
| CSR_QRR | 不变 | 不变 | 不变 | 不变 | 天然不变（基于距离比值） |
| CSR_TRR | 不变 | 不变 | 不变 | **会变** | 评估两种手性取较优 |
| NRMS | — | — | — | **会变** | Procrustes 对齐含反射 |
| Kendall tau | 不变 | 不变 | 不变 | 不变 | 天然不变（基于距离排序） |
| K_geom | 不变 | 不变 | 不变 | 不变 | 在规范坐标系中计算 |

## 流程图

```
VLM 回答
  │
  ├─ QRR 存在矛盾环？──是──→ infeasible（无 positions，无 SVG）
  │
  ├─ 对象 < 3 个？──────是──→ underconstrained（无 SVG）
  │
  └─ 通过 → Solver 求解（L-BFGS-B, 10 次重启）
              │
              ├─ 无解 → infeasible（无 SVG）
              │
              └─ 有解 → 生成 SVG
                        │
                        ├─ CSR_QRR < 0.95 或 CSR_TRR < 0.95
                        │   → infeasible（有 SVG，但重建失败）
                        │
                        └─ 两项 CSR ≥ 0.95
                            → feasible（重建成功）
                            → 细分为 single_mode / multimodal / underconstrained
```
