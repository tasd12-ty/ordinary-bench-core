# center_table_livingroom_v3

Checked-in reference example for the current Infinigen indoor prototype.

Contents:

- `view_0.png` to `view_3.png`
  - four benchmark-style orbit views around a large center table
- `metadata.json`
  - table size, object placement, and per-view camera metadata

This example was produced from a real Infinigen `LivingRoom` coarse scene and
then restaged with:

- no desk lamp
- a large center table
- four probe objects arranged according to the relative layout of
  `data-gen/output/scenes/n04_000000.json`
