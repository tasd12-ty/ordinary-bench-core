# 如何科学严谨地测量 VLM 的外显空间场景表征

> 实验实现规格文档 — ORDINARY-BENCH

## Context

当前 forced batch QA 方案接近 random baseline，原因是三种状态被压成一种输出：
- 真正看见了 → 正确答案
- 没看见但被迫猜 → 噪声
- 看见了但模糊 → 被强制二值化的信息丢失

目标不是"提高准确率"，而是：**从 VLM 的行为中反演出它可外显的空间场景信念，并科学地证明这个信念确实来自视觉感知而非语言先验。**

---

## 1. 核心科学问题

不是"VLM 答对了多少题"，而是：

> **VLM 的空间回答中，有多少信息来自它真正看到的图像，而不是语言先验？**

要回答这个问题，需要三样东西：

1. **信号**：VLM 看到正确图像后给出的约束集 C_visual
2. **噪声基线**：VLM 不看图像（或看错图像）时给出的约束集 C_null
3. **比较方法**：C_visual 比 C_null 好多少？这个"好"要可量化、可检验

---

## 2. 三条件实验设计（科学严谨性的基础）

| 条件 | 输入 | 含义 |
|------|------|------|
| **A: 正常** | VLM 看正确图像 + 回答空间问题 | 视觉+语言信号 |
| **B: 错误图像** | VLM 看另一个场景的图像 + 回答原场景问题 | 语言先验 + 视觉干扰 |
| **C: 无图像** | VLM 只看文字问题（无图像） | 纯语言先验 |

**文献支撑**：
- MMStar (NeurIPS 2024): GeminiPro 无图像达 42.9%，比 random 高 20%+
- VLind-Bench (NAACL 2025): 除 GPT-4o 外所有 VLM 严重依赖语言先验
- Counterfactual VQA (CVPR 2021): 反事实推断分离语言效应的标准方法

**对每个条件做完整重建**：
- S_A = reconstruct(C_A)（应接近 GT）
- S_B = reconstruct(C_B)（不应接近 GT）
- S_C = reconstruct(C_C)（不应接近 GT）

**视觉信息增益**：
```
ΔRecon = d(S_C, GT) - d(S_A, GT)
```
ΔRecon > 0 ⟹ 图像确实贡献了空间信息（不只是语言先验）

**统计检验**：
- 置换检验：随机打乱图像-问题配对，重建 N 次，计算 p-value
- 如果正确配对下的重建质量显著优于随机配对 → 视觉信息真实存在

---

## 3. 约束可靠性的客观测量（不依赖 VLM 自我报告）

VLM 自报置信度 calibration 很差（Groot 2024: 普遍过度自信）。需要客观方法。

### 3.1 一致性探测（Khan et al. CVPR 2024）

对同一空间关系问 K 次，每次换措辞：

```
Q1: "A-B 的距离和 C-D 的距离，哪个更大？"
Q2: "如果从 A 走到 B，再从 C 走到 D，哪段路更短？"
Q3: "A 离 B 近还是 C 离 D 近？"
```

一致性分数 = K 次中答案相同的比例。

**关键发现**（Khan et al.）：即使 rephrasing 不完全语义等价，一致性仍然与正确率强相关。而且此方法对黑盒 API 模型有效。

### 3.2 传递一致性（结构检验）

如果 VLM 说 d(A,B) < d(C,D) 且 d(C,D) < d(E,F)：
- 它是否也说 d(A,B) < d(E,F)？
- 违反传递性 = 至少一个回答是噪声

传递违反率 T_viol 是约束集质量的结构指标。
- 语言先验产生的回答不太可能满足传递性（因为不基于统一的空间模型）
- 真实视觉感知产生的回答应该近似传递

### 3.3 TRR 互逆一致性

如果 VLM 说 "A 在 B→C 的 3 点钟方向"：
- 那么 "A 在 C→B 的方向" 应该是 9 点钟（反向）
- 违反 = 回答不基于统一空间表征

**这三个测量（rephrasing 一致性、传递一致性、互逆一致性）都不依赖 GT，不依赖 VLM 自我报告，完全客观。**

---

## 4. 联合异常检测 + 嵌入（Ma et al. AAAI 2019）

这是最关键的数学工具。不是先筛选再重建（两阶段），而是联合求解：

```
y_c = d_ij - d_lk + γ_c + ε_c
```

其中：
- y_c：VLM 给出的比较方向
- d_ij - d_lk：真实距离差（来自待求的嵌入 X）
- γ_c：异常指示器（稀疏，L1 正则）— 标识哪些回答是错误的
- ε_c：高斯噪声 — 模型不确定性

**优化目标**：
```
min_{G, γ} L_w(G, γ) + λ ||γ||_{1,w}
s.t. rank(G) = p, G ≥ 0
```

**为什么这比加权方法更严谨**：
- 加权方法：人为指定权重，没有理论保证
- Ma et al.：权重（γ）由数据联合学习，全局一致性约束（SDP）保证几何合理性
- 最终输出不只是嵌入，还有每条约束的异常指标 → 知道哪些回答是错的

**对我们的意义**：
- 不需要预先知道哪些 VLM 回答是对的
- 算法自动发现哪些是 outlier
- outlier 集合本身就是有价值的诊断信息

---

## 5. 具体的提问协议设计

### 5.1 半结构化三级输出

**Level 1: 确定性选择**
```
Q: A-B 和 C-D，哪对距离更大？
选项: [A-B更大 / C-D更大 / 差不多 / 无法判断]
```

**Level 2: 集合式答案**
```
Q: A 相对于 B→C 的方向，最接近钟表几点？
选项: [可多选 1-12 + "无法判断"]
```

**Level 3: 一致性验证（同一关系换说法问 K=3 次）**

### 5.2 为什么不用 free-form

- 解析噪声太大（VLM 输出格式不可控）
- 无法做传递/互逆一致性检查
- 无法与 null model 做可比较的实验

半结构化的好处：
- 答案格式固定 → 可自动解析
- 允许模糊（多选、象限粒度）→ 保留不确定性信息
- 允许拒答 → 区分"不知道"和"猜错"
- 可换措辞重问 → 客观测量一致性

## 6. 约束分级与映射

从 VLM 回答到约束的映射：

| VLM 回答 | 约束类型 | 在 solver 中的处理 |
|---------|---------|----------------|
| "A-B 更大"，一致性 3/3 | hard QRR `>` | 权重 1.0 |
| "A-B 更大"，一致性 2/3 | soft QRR `>` | 权重 0.67 |
| "A-B 更大"，一致性 1/3 | weak QRR `>` | 权重 0.33 或丢弃 |
| "差不多" | QRR `~=` | 正常处理 |
| "无法判断" | **不产生约束** | 不参与 loss |
| TRR 选了单 hour | hard TRR, tol=15° | 权重 1.0 |
| TRR 选了 2-3 个 hour | soft TRR, tol=30°-45° | 中间宽度 |
| TRR 选了象限 | coarse TRR, tol=45° | 宽弧段 |
| TRR "无法判断" | **不产生约束** | 不参与 loss |

## 7. 完整评估指标体系

### 7.1 无需 GT 的结构指标

| 指标 | 定义 | 含义 |
|------|------|------|
| **拒答率** r_abstain | "无法判断" 占比 | VLM 的自知之明 |
| **一致性分数** c_mean | 平均 rephrasing 一致性 | 回答可靠性 |
| **传递违反率** T_viol | 传递关系违反比例 | 约束集结构质量 |
| **互逆违反率** R_viol | TRR 互逆关系违反比例 | 方向感知一致性 |
| **CSR** | 重建满足约束比例 | 约束可满足性 |
| **K_geom** | 几何模态数 | 解的唯一性 |
| **spread** | 解空间大小 | 约束强度 |
| **outlier 比例** \|γ\|/\|C\| | 联合检测到的异常约束比例 | 噪声水平 |

### 7.2 需要 GT 的验证指标

| 指标 | 定义 | 含义 |
|------|------|------|
| **NRMS** | 归一化 RMS 误差 | 绝对重建精度 |
| **Kendall τ** | 距离序相关 | 序保持程度 |
| **ΔRecon** | d(S_null, GT) - d(S_visual, GT) | 视觉信息增益 |
| **信息比** | ΔRecon / d(S_null, GT) | 相对视觉贡献 |

### 7.3 科学解释性的关键：null model 对比

```
如果 ΔRecon ≈ 0 → VLM 的空间回答不含视觉信息（全是语言先验）
如果 ΔRecon > 0 且 p < 0.05 → VLM 确实从图像中提取了空间信息
ΔRecon 的大小 = VLM 外显空间感知的信息量
```

## 9. 这套框架能回答什么、不能回答什么

### 能回答
- VLM 从图像中外显出的空间关系集合是什么
- 这些关系中哪些是可靠的（一致性高）、哪些是噪声
- 从可靠关系反演出的场景是紧的还是松的、单模态还是多模态
- VLM 的空间信息有多少来自视觉、多少来自语言先验（ΔRecon）
- VLM 对哪些物体/关系类型有真正的视觉感知、哪些完全依赖猜测

### 不能回答
- VLM 内部 attention head 在做什么（这是 mechanistic interpretability）
- VLM 是否"真正理解"空间（哲学问题）
- VLM 的内部表征几何结构（需要 probing hidden states）
- 视觉感知 vs 语言幻觉的精确边界（存在不可辨识区间）

### 方法论边界
- 一致性高 ≠ 一定来自视觉（可能是稳定的语言先验）→ 必须配合 null model
- 拒答多 ≠ VLM 真不知道（可能只是 prompt 让它保守）→ 需要控制 prompt 的拒答率
- outlier 检测依赖"多数正确"假设 → 如果 >50% 是错的，Ma et al. 方法会失效
